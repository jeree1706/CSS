1- ¿Qué significa CSS?

CSS significa en ingles: "Cascading Style Sheets", que traducido: "Hojas de Estilo en Cascada"

2- ¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

Sirve para darle un estilo más bonito a la página web. El programa que se utiliza es el navegador

3- ¿A qué se llama regla de CSS?

Se le llama regla de CSS al conjunto de propiedades y valores que se aplicarán a una o varias etiquetas

4- ¿Cuál es la anatomía de la regla de CSS? Agregar imagen explicativa y utilizando una lista descriptiva explicar sus elementos.

La anatomía de la regla de CSS es:

Selector
Selecciona el elemento a estilizar
Declaración
Aplica las propiedades que querés dar al elemento seleccionado en el Selector
Propiedad
Es lo que te permite dar un estilo a un elemento HTML
Valor de propiedad
Es el valor definido que tiene la propiedad

5- ¿Qué es un selector en CSS?

Un selector en CSS es lo primero que se escribe en la regla del CSS. Indican al navegador qué elementos HTML se seleccionan para aplicarles una regla

6- Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

De ID: Permite que el selector se utilice unicamente una vez, se indica con el #
De clases: Permite que el selector se utilice como minimo 1 vez, y sin limite. Se indica con el .
Universal: Permite aplicar estilos a todos los elementos HTML de un documento y se indica mediante el *
De etiquetas: Permite aplicar propiedades CSS a una etiqueta. La misma se identifica sin utilizar "<>"
Descendente: Con este seleccionamos un elemento que está dentro de otro del mismo tipo. El último selector será el elemento HTML al que se le aplicará las propiedades de CSS.

7- Explicar brevemente utilizando una lista ordenada: ¿cómo funciona el CSS?

  1. El navegador carga el HTML
  2. El HTML se convierte en un DOM
  3. El navegador busca la mayor parte de los recursos vinculados al documento HTML, y además el CSS vinculado
  4. El navegador analiza el CSS y ordena en diferentes «cubos» las diferentes reglas según el tipo de selector, dependiendo el selector calcula qué reglas deben aplicarse y a qué nodos en el DOM se les aplica el estilo según corresponda
  5. El árbol de renderización presenta la estructura en que los nodos deben aparecer después de aplicarle las reglas.
  6. En la pantalla se muestra el aspecto visual de la página

8- ¿Cuáles son las tres formas para vincular CSS con HTML?

Las tres formas de vincular CSS con HTML son:
-Se usa la etiqueta "style" en el head del HTML
- Se usa el atributo "style" dentro de alguna etiqueta del HTML
- Crear un nuevo archivo CSS y vincularlo con el HTML
